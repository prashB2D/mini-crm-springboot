# Mini CRM - Spring Boot Application

A Customer Management System built using Spring Boot, JSP, Hibernate, and MySQL.

## Features
- Add Customer
- View Customer
- Update Customer
- Delete Customer

## Tech Stack
- Java 17
- Spring Boot
- JSP
- Hibernate / JPA
- MySQL
- HTML / CSS

## How to Run

1. Clone the repository
2. Create MySQL database
3. Update application.properties
4. Run as Spring Boot Application

Open in browser:
http://localhost:8080/CRM/show-form

Database Name: miniCRM
Port: 8080
Context Path: /CRM
